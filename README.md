## 🔍 О проекте

CryptoTrader v2.0 — это эволюция от «парсера команд» к **автономному AI-стратегу**.

### Ключевое изменение

| v1.0 | v2.0 |
|------|------|
| LLM — исполнитель, не стратег | **DeepSeek — главный стратег и трейдер** |
| Пользователь пишет команды | Пользователь может делегировать торговлю AI |
| Только ручной режим | Ручной / Полуавтомат / Автомат |
| CoinEx API v2 с ошибками | **100% соответствие CoinEx API v2** |
| Только Linux | **Linux + Windows + macOS** |
| Лимиты: $1000/ордер, $200/день | **Все лимиты сняты** |

### Философия

DeepSeek выступает в роли **профессионального криптовалютного брокера** — ответственного управляющего, который относится к деньгам клиента так же бережно, как к своим. Он анализирует рынок, ищет возможности, управляет риском и не торгует ради торговли. Пользователь выбирает уровень автономности — от ручного режима до полностью автоматической торговли.

## 🆕 Что нового в v2.0

- **DeepSeek-брокер**: профессиональный управляющий, серьёзный и ответственный
- **Три режима**: MANUAL → SEMI_AUTO → FULL_AUTO
- **Чат с брокером**: общение с AI-брокером, приём торговых поручений на естественном языке
- **Авто-режим без подтверждений**: в FULL_AUTO сделки исполняются сразу после валидации
- **Все лимиты сняты**: без ограничений на размер ордера, дневной убыток, количество позиций, circuit breaker
- **Инструкции в авто-режиме**: можно давать команды через чат, пока стратег торгует в фоне
- **CoinEx API v2 — 100% совместимость**:
  - Правильная HMAC-SHA256 подпись (без `\n`, lowercase hex)
  - Все эндпоинты v2 (`/spot/order`, `/assets/spot/balance`, `/spot/ticker`)
  - Обязательный `market_type: "SPOT"`
- **Кроссплатформенность**: Linux, Windows (MSVC/MinGW), macOS

## 📦 Установка и сборка

### Linux (Ubuntu/Debian)

```bash
sudo apt update && sudo apt install -y \
    cmake g++ libssl-dev libsqlite3-dev \
    libcurl4-openssl-dev nlohmann-json3-dev

cd crypto_trader && mkdir build && cd build
cmake .. && make -j$(nproc)
./crypto_trader
```

### Windows

```bash
# vcpkg
vcpkg install openssl curl sqlite3 nlohmann-json

# Или MSYS2/MinGW
pacman -S mingw-w64-x86_64-cmake mingw-w64-x86_64-gcc \
         mingw-w64-x86_64-openssl mingw-w64-x86_64-curl \
         mingw-w64-x86_64-sqlite3 mingw-w64-x86_64-nlohmann-json

mkdir build && cd build
cmake .. -G "MinGW Makefiles" && make -j$(nproc)
```

### macOS

```bash
brew install cmake openssl curl sqlite3 nlohmann-json
mkdir build && cd build
cmake .. && make -j$(sysctl -n hw.ncpu)
```

## 🎮 Использование

### Запуск

```bash
./crypto_trader                          # DRY-RUN, ручной режим
./crypto_trader --live --manual          # LIVE, ручной режим
./crypto_trader --semi-auto              # Стратег-советник (DRY-RUN)
./crypto_trader --auto                   # Авто-стратег (DRY-RUN)
./crypto_trader --live --auto            # Авто-стратег LIVE (осторожно!)
```

### Чат с брокером

DeepSeek-брокер общается профессионально, сдержанно и по делу:

```
crypto[DRY]> привет
AI: Добрый день. Я ваш брокер. Готов помочь с торговлей на CoinEx.

crypto[DRY]> что думаешь про BTC?
AI: BTC в восходящем тренде, объёмы выше среднего. Уровень сопротивления — $66000. При пробое — потенциал до $68000. Риск контролируемый.

crypto[DRY]> купи 0.1 ETH по рынку, тейк 10%
AI: Исполняю: покупка 0.1 ETHUSDT по рынку.
⚡ Обнаружена торговая команда: buy 0.1 ETHUSDT
```

### Системные команды

| Команда | Действие |
|---------|----------|
| `баланс` / `balance` | Баланс и стоимость портфеля |
| `статус` / `status` | Статус, режим, статистика |
| `история` / `history` | Последние 15 ордеров |
| `лимиты` / `limits` | Параметры стратегии |
| `стратег` / `strategist` | Включить полуавтомат |
| `авто` / `auto` | Включить авто-стратег |

| `помощь` / `help` | Справка |
| `выход` / `exit` | Выход |

## 🧠 Режимы стратега

### MANUAL (Ручной)
Пользователь отдаёт торговые поручения или общается с AI-брокером в чате. Брокер парсит поручения в ордера или отвечает на вопросы — профессионально и по делу. Валидатор проверяет баланс, биржа исполняет (DRY-RUN симулирует с обновлением баланса).

### SEMI_AUTO / FULL_AUTO (Авто-стратег)
Брокер торгует **полностью самостоятельно**. Каждые 5 минут:
1. Получает контекст рынка (цены, баланс, позиции, история, P&L)
2. Анализирует: тренд, риск/прибыль, история, момент для входа
3. Принимает решение — **buy/sell/hold/cancel**
4. Исполняет немедленно, без подтверждений

**Брокер никого не ждёт.** Он не спрашивает разрешения. Его задача — торговать.

Клиент может дать поручение в любой момент — брокер исполнит его мгновенно и продолжит автономную торговлю.

## 🔑 API ключи

### DeepSeek

1. [platform.deepseek.com](https://platform.deepseek.com) → API Keys
2. Создать ключ (формат: `sk-...`)
3. Пополнить баланс (~$0.14/1M токенов)

```bash
export DEEPSEEK_API_KEY="sk-..."
```

### CoinEx

1. [coinex.com](https://www.coinex.com) → Security → API Management
2. Разрешения: **Spot Trading, Spot Order Query, Asset Query**
3. Отключить: Futures, Withdraw

```bash
export COINEX_API_KEY="XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX"
export COINEX_API_SECRET="xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

## 📡 CoinEx API v2

Проект полностью совместим с [официальной документацией CoinEx API v2](https://docs.coinex.com/api/v2/).

### Аутентификация (HMAC-SHA256)

```
prepared_str = METHOD + request_path[?query] + body + timestamp
signature = HMAC-SHA256(secret_key, prepared_str) → lowercase hex
```

Заголовки: `X-COINEX-KEY`, `X-COINEX-SIGN`, `X-COINEX-TIMESTAMP`.

### Используемые эндпоинты

| Метод | Эндпоинт | Назначение |
|-------|----------|-----------|
| `GET` | `/assets/spot/balance` | Баланс спотового аккаунта |
| `GET` | `/spot/ticker?market=X,Y` | Тикеры (до 10 пар) |
| `GET` | `/spot/market` | Список рынков |
| `POST` | `/spot/order` | Создание ордера |
| `GET` | `/spot/pending-order` | Открытые ордера |
| `GET` | `/spot/finished-order` | Завершённые ордера |
| `DELETE` | `/spot/order` | Отмена ордеров |

Все значения `amount`/`price` передаются и принимаются как **строки** (спецификация API v2).

## ❓ FAQ

**Q: DeepSeek может потерять все деньги?**
A: DeepSeek запрограммирован как ответственный брокер: сохранять капитал — его первая заповедь. В DRY-RUN риска нет (симуляция). В LIVE — только спот без плеч, валидация каждого ордера, и брокер не входит в сделку без чёткого сигнала.

**Q: Какая комиссия CoinEx?**
A: 0.2% за спот-сделку. Для VIP — ниже.

**Q: Сколько стоит DeepSeek?**
A: ~$0.14 за 1M токенов. Один анализ рынка (~2000 токенов) стоит ~$0.0003. $1 хватит на ~3000 итераций стратега.

**Q: Какие пары поддерживаются?**
A: По умолчанию BTCUSDT, ETHUSDT, SOLUSDT. Можно добавить любые спот-пары CoinEx.

**Q: Можно использовать без DeepSeek?**
A: Системные команды (баланс, статус) работают без ключа. Чат и стратег требуют ключ.

**Q: Где хранятся данные?**
A: `crypto_trader.db` (SQLite) и `trades.jsonl` (JSON-лог) в папке запуска.
