## 🔍 О проекте

CryptoTrader v2.0 — это эволюция от «парсера команд» к **автономному AI-стратегу**.

### Ключевое изменение

| v1.0 | v2.0 |
|------|------|
| LLM — исполнитель, не стратег | **DeepSeek — главный стратег и трейдер** |
| Пользователь пишет команды | Пользователь может делегировать торговлю AI |
| Только ручной режим | Ручной / Полуавтомат / Автомат |
| CoinEx API v2 с ошибками | **100% соответствие CoinEx API v2** |
| Только Linux | **Linux + Windows + macOS** |

### Философия

DeepSeek получает полный контекст рынка (цены, баланс, позиции, открытые ордера, P&L) и самостоятельно принимает торговые решения в рамках заданных лимитов. Пользователь выбирает уровень автономности.

## 🆕 Что нового в v2.0

- **DeepSeek-стратег**: AI анализирует рынок и торгует сам
- **Три режима**: MANUAL → SEMI_AUTO → FULL_AUTO
- **CoinEx API v2**: полное соответствие официальной документации
  - Правильная HMAC-SHA256 подпись (без `\n`, lowercase hex)
  - Все эндпоинты v2 (`/spot/order`, `/assets/spot/balance`, `/spot/ticker`)
  - Обязательный `market_type: "SPOT"`
- **Кроссплатформенность**: Linux, Windows (MSVC/MinGW), macOS
- **Улучшенный контекст рынка**: тикеры, объёмы, позиции, P&L
- **Стратегия с лимитами**: настраиваемые уровни риска

## 📦 Установка и сборка

### Linux (Ubuntu/Debian)

```bash
# Зависимости
sudo apt update && sudo apt install -y \
    cmake g++ libssl-dev libsqlite3-dev \
    libcurl4-openssl-dev nlohmann-json3-dev

# Сборка
cd crypto_trader
mkdir build && cd build
cmake ..
make -j$(nproc)

# Запуск
./crypto_trader
```

### Windows

```bash
# Вариант 1: vcpkg (рекомендуется)
vcpkg install openssl curl sqlite3 nlohmann-json

# Вариант 2: MSYS2/MinGW
pacman -S mingw-w64-x86_64-cmake mingw-w64-x86_64-gcc \
         mingw-w64-x86_64-openssl mingw-w64-x86_64-curl \
         mingw-w64-x86_64-sqlite3 mingw-w64-x86_64-nlohmann-json

# Сборка
mkdir build && cd build
cmake .. -G "MinGW Makefiles"
make -j$(nproc)
```

### macOS

```bash
brew install cmake openssl curl sqlite3 nlohmann-json
mkdir build && cd build
cmake ..
make -j$(sysctl -n hw.ncpu)
```

## 🎮 Использование

### Запуск

```bash
# DRY-RUN, ручной режим (по умолчанию)
./crypto_trader

# LIVE, ручной режим
./crypto_trader --live --manual

# Стратег-советник (предлагает сделки, вы подтверждаете)
./crypto_trader --semi-auto

# Авто-стратег (торгует сам в DRY-RUN)
./crypto_trader --auto

# Авто-стратег в LIVE (осторожно!)
./crypto_trader --live --auto
```

### Команды в программе

#### Ручной режим (MANUAL)

```
crypto[DRY]> купи 0.15 ETH по рынку, тейк 10%, стоп 3%
crypto[DRY]> продай 0.5 BTC по лимиту 65000
crypto[DRY]> buy 100 USDT of SOL
```

#### Системные команды

| Команда | Действие |
|---------|----------|
| `баланс` / `balance` | Показать баланс и стоимость портфеля |

## 🧠 Режимы стратега

### MANUAL (Ручной)
Пользователь пишет торговые команды, DeepSeek преобразует их в структурированный формат. Валидатор проверяет риски, биржа исполняет.

### SEMI_AUTO (Стратег-Советник)
DeepSeek запускается в цикле (по умолчанию каждые 5 минут):
1. Получает контекст рынка (цены, баланс, позиции)
2. Анализирует и формирует торговое решение
3. Показывает решение пользователю с объяснением
4. Пользователь подтверждает (yes/no/skip)
5. Исполнение на бирже

```
╔══════════════════════════════════════════╗
║        РЕШЕНИЕ СТРАТЕГА (DeepSeek)       ║
╚══════════════════════════════════════════╝
  Действие:    ПОКУПКА
  Пара:        ETHUSDT
  Объём:       0.15
  Тип:         market
  Тейк:        +5%
  Стоп:        -3%
  Уверенность: 85%
  Стратегия:   трендовая
  Причина:     ETH пробил уровень сопротивления...

  Подтвердить? (yes/no/skip):
```

### FULL_AUTO (Авто-Стратег)
DeepSeek торгует **полностью самостоятельно**:
- Анализ рынка каждые 5 минут
- Автоматическое принятие решений
- Соблюдение всех лимитов безопасности
- Требует минимальную уверенность 75% для сделки
- В LIVE требует дополнительное подтверждение при запуске

## 🔑 API ключи

### DeepSeek

1. [platform.deepseek.com](https://platform.deepseek.com) → API Keys
2. Создать ключ (формат: `sk-...`)
3. Пополнить баланс (~$0.14/1M токенов)

```bash
export DEEPSEEK_API_KEY="sk-..."
```

### CoinEx

1. [coinex.com](https://www.coinex.com) → Security → API Management
2. Создать ключ с разрешениями: **Spot Trading, Spot Order Query, Asset Query**
3. Отключить: Futures, Withdraw

```bash
export COINEX_API_KEY="XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX"
export COINEX_API_SECRET="xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```

## 📡 CoinEx API v2

Проект полностью совместим с [официальной документацией CoinEx API v2](https://docs.coinex.com/api/v2/).

### Аутентификация (HMAC-SHA256)

```
prepared_str = METHOD + request_path[?query] + body + timestamp
signature = HMAC-SHA256(secret_key, prepared_str) → lowercase hex
```

Заголовки: `X-COINEX-KEY`, `X-COINEX-SIGN`, `X-COINEX-TIMESTAMP`.

### Используемые эндпоинты

| Метод | Эндпоинт | Назначение |
|-------|----------|-----------|
| `GET` | `/assets/spot/balance` | Баланс спотового аккаунта |
| `GET` | `/spot/ticker?market=X,Y` | Тикеры (до 10 пар) |
| `GET` | `/spot/market` | Список доступных рынков |
| `POST` | `/spot/order` | Создание ордера |
| `GET` | `/spot/pending-order` | Открытые ордера |
| `GET` | `/spot/finished-order` | Завершённые ордера |
| `DELETE` | `/spot/order` | Отмена ордеров |

Все значения `amount`/`price` передаются и принимаются как **строки** (спецификация API v2).

## 🛡 Безопасность

| Механизм | Описание |
|----------|----------|
| **DRY-RUN по умолчанию** | Реальная торговля только с `--live` + подтверждение |
| **Circuit breaker** | 3 ошибки → 5 минут паузы |
| **Лимит ордера** | Макс. $1000 на ордер (настраивается) |
| **Дневной лимит** | Макс. $200 убытка в день (настраивается) |
| **Мин. уверенность** | Авто-сделки только при confidence ≥ 75% |
| **Только SPOT** | Фьючерсы и маржа заблокированы |
| **Валидация** | Проверка каждого ордера перед отправкой |
| **Аудит** | Полное логирование в SQLite + JSONL |

## ❓ FAQ

**Q: DeepSeek может потерять все деньги?**
A: Нет. Авто-стратег работает в рамках жёстких лимитов: максимальный ордер, дневной лимит убытков, минимальная уверенность. Circuit breaker останавливает торговлю при ошибках. В LIVE требует подтверждения.

**Q: Какая комиссия CoinEx?**
A: 0.2% за спот-сделку. Для VIP — ниже.

**Q: Сколько стоит DeepSeek?**
A: ~$0.14 за 1M токенов. Один анализ рынка (~2000 токенов) стоит ~$0.0003. $1 хватит на ~3000 итераций стратега.

**Q: Какие пары поддерживаются?**
A: По умолчанию BTC/USDT, ETH/USDT, SOL/USDT. Можно добавить любые спот-пары CoinEx.

**Q: Можно использовать без DeepSeek?**
A: Системные команды (баланс, статус) работают без ключа. Для торговли и стратега нужен ключ.

**Q: Где хранятся данные?**
A: `crypto_trader.db` (SQLite) и `trades.jsonl` (JSON-лог) в папке запуска.
